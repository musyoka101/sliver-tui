# Theme System - Quick Reference

## Available Themes (Press 't' to cycle)

**14 Themes Available:**

1. **Default (Dracula)** - Purple and cyan inspired by Dracula theme
   - Cyan title, purple logo
   - Green sessions, yellow beacons, gray dead
   - Red privileged, green normal users
   - Cyan protocols, dark background tints

2. **Rainbow** - Vibrant multi-color palette
   - Magenta title, pink logo
   - Bright green sessions, orange beacons, red dead
   - Gold privileged, cyan normal users
   - Purple MTLS, blue HTTP, green DNS, pink TCP, yellow default

3. **Cyberpunk** - Neon aesthetic (pinks, purples, electric blues)
   - Neon pink title, purple logo
   - Electric blue sessions, hot pink beacons, dark purple dead
   - Bright cyan privileged, neon yellow normal users
   - Neon green protocols with cyberpunk glow

4. **Matrix** - Classic green terminal (iconic hacker aesthetic)
   - Bright green title and logo
   - Matrix green sessions, yellow-green beacons, dark green dead
   - Gold privileged accents, light green normal users
   - All green protocols with matrix tints

5. **Tactical** - Military colors (olive, orange, steel)
   - Orange title, olive green logo
   - Green sessions, yellow beacons, red dead
   - Gold privileged, cyan normal users
   - Steel blue MTLS, teal HTTP, orange DNS, olive TCP

6. **Pastel** - Soft colors (easy on eyes, gentle aesthetic)
   - Soft pink title, lavender logo

   - Soft blue protocols, pastel tints

7. **Heatmap** - Priority-based (red=high, blue=low)
   - White title, red logo

   - Red privileged, orange normal users
   - Red MTLS, orange HTTP, yellow DNS, blue TCP, heat gradient

8. **Lip Gloss** - Charm's signature purple/pink/blue gradients

9. **Nord** - Popular Nordic cool blues and grays

10. **Gruvbox** - Retro warm color scheme

13. **Catppuccin Mocha** - Warm, cozy dark theme (soothing pastel colors)

## Controls

- **'t'** - Cycle through themes
- **'r'** - Refresh data
- **'↑↓'** or **'j/k'** - Scroll
- **'q'** - Quit
