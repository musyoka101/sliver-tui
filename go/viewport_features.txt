TECHNICAL IMPLEMENTATION:
==========================

1. **Viewport Initialization**
   - Created on first tea.WindowSizeMsg
   - Calculates height: termHeight - headerFooter
   - Sets Y position to start after header

2. **Content Updates**
   - updateViewportContent() called on new agents
   - Renders agent list with logo
   - Sets viewport.SetContent()
   - Preserves scroll position

3. **Scroll Handling**
   - Keyboard events passed to viewport.Update()
   - Viewport handles all scroll logic
   - Returns scroll commands for smooth animation

4. **Position Tracking**
   - viewport.ScrollPercent() gives 0.0 to 1.0
   - Displayed as percentage in header
   - Updates every frame during scroll
