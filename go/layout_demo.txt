HOW IT WORKS:
=============

1. Terminal Width Detection:
   - Captures actual terminal width via tea.WindowSizeMsg
   - Updates dynamically when terminal is resized
   - Works with any terminal size (80 cols to 200+ cols)

2. Dynamic Spacing Calculation:
   Formula: spacing = termWidth - mainContentWidth - panelWidth - 2

   Example with 180 char terminal:
   - Main content width: ~100 chars (agent list)
   - Panel width: 39 chars (35 content + 4 border)
   - Spacing: 180 - 100 - 39 - 2 = 39 chars
   
   The panel ALWAYS sticks to the far right!

3. Responsive Behavior:
   âœ… Wide terminals (180+): Panel hugs right edge with large gap
   âœ… Medium terminals (120-180): Panel at right with moderate gap
   âœ… Narrow terminals (80-120): Panel still at right with minimum 2-char gap
   âœ… Resize: Panel repositions automatically

4. Benefits:
   - Professional operator feel
   - Panel doesn't interfere with main content
   - Easy to glance at tactical intel
   - Clean visual separation
   - Maximizes use of screen real estate
